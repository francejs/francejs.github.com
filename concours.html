Les défis FranceJS

Vous aimez quand il y a du sang sur les murs ? Alors les défis FranceJS sont faits pour vous. Le principe est simple. Des questions vous sont posées par le biais de la console et vous devez les résoudre pour remporter le défi.

Les défis :

  • Défi de Nicolas Froidure ;
  • Défi de Matthias Le Brun.

Pour proposer votre défi, faîtes un pull request avec un fichier HTML nommé defi-(screename).html.
